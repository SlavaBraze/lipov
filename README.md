## Описание программы

**Weather Diary** - это графическое приложение с дружественным интерфейсом для ведения персонального дневника погоды. Программа позволяет пользователям добавлять, просматривать, редактировать и удалять записи о погодных условиях, а также фильтровать их по различным критериям.

### Основные возможности

1. **Управление записями:**
   - Добавление новых записей о погоде
   - Редактирование существующих записей
   - Удаление отдельных записей
   - Полная очистка дневника

2. **Хранение данных:**
   - Автоматическое сохранение в JSON файл
   - Постоянное хранение в одном файле (`weather_data.json`)
   - Автосохранение при любых изменениях
   - Загрузка данных при запуске программы

3. **Фильтрация записей:**
   - По точной дате (ГГГГ-ММ-ДД)
   - По диапазону дат (от... до...)
   - По диапазону температур (мин. и макс. температура)
   - По наличию осадков (Да/Нет)
   - Комбинирование нескольких фильтров одновременно

4. **Статистика:**
   - Общее количество записей
   - Средняя температура
   - Минимальная и максимальная температура
   - Количество и процент дней с осадками

5. **Валидация данных:**
   - Проверка формата даты
   - Проверка корректности температуры (должно быть число)
   - Проверка заполнения описания
   - Защита от дублирования записей за одну дату

## Технические требования

### Системные требования
- **Операционная система:** Windows / Linux / macOS
- **Python версия:** 3.7 или выше
- **Оперативная память:** от 128 MB
- **Свободное место на диске:** от 10 MB

### Используемые библиотеки
Все библиотеки являются стандартными и входят в состав Python:
- `tkinter` - для создания графического интерфейса
- `json` - для работы с JSON файлами
- `datetime` - для работы с датами
- `os` - для работы с файловой системой
